# Sublime Text Config

`sublime-text-config` contains my personal config files for [Sublime Text](https://www.sublimetext.com/).

### Usage

```bash
cd ~/.config/sublime-text-3/Packages
rm -rf User/
git clone https://github.com/trey-c/sublime-text-config.git User
```

### Requirements

- clang
- clang-format
- clang-tidy
- git
